This mod fixes all of the soldier animations so that they (both male and females of XCom and the various Faction Heros) grip any rifles that don't have pistol grips properly, instead of looking wack. It's configurable and other mods can make this a requirement if they need their soldiers to "Get A Rifle Grip" for their weapon.

This mod fixes the following weapons from the Resistance Firearms weapon pack:
[list]
     [*]M14 - Classic
     [*]M14 with KAC Railkit
	 [*]M21
	 [*]R700
[/list]

This mod fixes the following weapons from the CoD IW weapon pack:
[list]
	 [*]EBR-800
	 [*]EBR-800 - Osiris (XBR-1)
	 [*]EBR-800 - Bomber (M8 Garand)
[/list]

[b]Q: What is true endgoal of this mod?[/b]

A: The endgoal of this mod was so that I can start porting other rifle grip weapons like the M1 Garand, Lee Enfield, Springfield M1903, BAR, PPsh-41, Mosin Nagant, Arisaka, Kbsp wz. M31, M1897 Trench Gun, etc. without the soldiers not holding the rifles properly. To be honest, it bugged me for a long time, until the day of this mod's release.

[b]Q: What is the point of this mod?[/b]

A: This mod is to fix some of the battle rifles and snipers that use rifle grips instead of pistol grips.

[b]Q: Vanilla version when?[/b]

A: Not really, no.

[b]Q: LWOTC/PP/etc etc etc when?[/b]
