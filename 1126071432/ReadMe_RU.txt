Модификация «Пистолеты-пулеметы» для игры XCOM2
Создатель: студия Long War

Данная модификация добавляет в игру три пистолета-пулемета для каждого из трех уровней оружия: огнестрельного, магнитного и плазменного. 

По сравнению с винтовкой, этот тип оружия обеспечивает лучшую маскировку, мобильность и меткость на ближних дистанциях ценой уменьшения урона и снижения меткости на дальних дистанциях.

----------------------------------

Самостоятельная установка модификации
1) Скопируйте содержимое модификации в папку \Steam\steamapps\common\XCOM 2\XComGame\Mods\
2) Добавьте в файл конфигурации \Steam\steamapps\common\XCOM 2\XComGame\Config\DefaultModOptions.ini строчку:  ActiveMods="LW_SMGPack"

Самостоятельное удаление модификации
1) Удалите из файла конфигурации \Steam\steamapps\common\XCOM 2\XComGame\Config\DefaultModOptions.ini строчку:  ActiveMods="LW_SMGPack"
2) Удалите папку \Steam\steamapps\common\XCOM 2\XComGame\Mods\LW_SMGPack\
