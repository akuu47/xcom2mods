This mod adds the Boss' Open sneaking suit that can be worn by females for all units from Metal Gear Solid V: The Phantom Pain (they were DLC for that game). This mod also comes with the snake scar as seen on the Boss' torso before the final battle at Rokovoj Bereg.

[h1][b]VANILLA VERSION: https://steamcommunity.com/sharedfiles/filedetails/?id=1359648885[/b][/h1]

[b]If anyone wants to help port the hero or any other uniforms, or accelerate the rate that this mod gets more content, feel free to get in touch with me. I really just need more skinners that are able to rig uniforms to XCOM 2.[/b]

If you have issues with this mod or spotted a bug, please report it in the [url=http://steamcommunity.com/workshop/filedetails/discussion/1330558644/1698293703767367022/]Bug Report Thread[/url]. If you want to provide feedback, whether its good or bad, then post in the [url=http://steamcommunity.com/workshop/filedetails/discussion/1330558644/1698293703767368694/]General Feedback Thread[/url].

If you are thinking of downvoting this mod, please provide feedback first, so I know what I should improve on/did wrong.

I don't ask for donations. Period.
