XCOM 2 LaserPack
Par Long War Studios

INSTRUCTIONS D'INSTALLATION
Steam Workshop : abonnez-vous au mod sur le Workshop de Long War Studios.

Manuelle/Nexus : ouvrez l'archive et copiez le répertoire LW_LaserPack ainsi que son contenu dans votre version du dossier C:\Program Files (x86)\Steam\steamapps\common\XCOM 2\XComGame\Mods.

Pour jouer :
Cochez le mod LW_LaserPack dans le Launcher. 

Ce mod ajoute les armes laser à impulsion, qui se situent, sur l'arbre des technologies, entre les armes magnétiques et les armes à rayons plasmiques. Ces armes octroient généralement un point de dégât supplémentaire par rapport aux armes magnétiques, ou 50% de chances de point de dégât supplémentaire pour les pistolets laser.

Pour débloquer la technologie laser à impulsion de base, vous devez avoir découvert l'élérium. Pour débloquer les lasers à impulsion avancés, vous devez avoir découvert la technologie laser à impulsion. Pour débloquer les armes (à rayons) plasmiques, vous devez désormais avoir découvert les lasers à impulsion avancés. Ceci retardera donc le moment où vous accéderez aux armes plasmiques, mais le coût en ressources pour le développement des lasers est comparable à celui des armes magnétiques, vous devriez donc pouvoir mettre la main sur les lasers relativement tôt.

Les armes laser incluent une mitraillette laser à impulsion, le pistolet-mitrailleur à impulsion, qui ne sera jouable que si vous avez installé le mod SMG de Long War Studios.
