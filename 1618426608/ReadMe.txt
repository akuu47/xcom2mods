This mod adds Sam Fisher's outfit from Splinter Cell: Blacklist available to wear for both males and females of all classes.

[quote]If anyone wants to help port other uniforms, or accelerate the rate of more content, feel free to get in touch with me. I really just need more skinners that are able to rig uniforms to XCOM 2.[/quote]

If you have issues with this mod or spotted a bug, please report it in the [url=link1]Bug Report Thread[/url]. If you want to provide feedback, whether its good or bad, then post in the [url=link2]General Feedback Thread[/url].

If you are thinking of downvoting this mod, please provide feedback first, so I know what I should improve on/did wrong.

I don't ask for donations. Period.
